Rate-It [github]

Python · Flask · JavaScript

Sentiment analysis web app to evaluate and categorize human sentiments within provided text. The interface prototype was built using Python, Flask, and JavaScript along with Python libraries like TextBlob to process plain text or CSV files. Currently refactoring the UI and backend to optimize for social post analysis through provided links.

Lyrify [github]

JavaScript · Bootstrap · React

Music lyric generation and search application developed with Spotify API playback using JavaScript and React. Since the initial development of this idea, Spotify has implemented its own lyric generation feature, affirming the need for one in any music playback app. I continue to iterate on this project, currently attempting to generate QR codes for users to share lyrics and timestamps.

Sonic Levels [github]

Java · Firebase

2D platformer game with stylized pixel art to match the aesthetic of major titles such as Mario and Sonic. This Java-based game draws inspiration from both and adds unique game experiences such as levels, in-game transactions, character customization, and limited power-ups.

RAIINET++ [github]

C++ · X11 GUI

Multiplayer strategy game named after the development concept of RAII (Resource Acquisition Is Initialization). Using that to optimize game generation as well as other OOP tenets, the game offers both text and graphical interfaces for players to compete in an engaging setting.

Auto Kart [github]

Arduino · C · Raspberry Pi

Autonomous robotic car built on a basis of Arduino robotics. While the initial prototype was built to quickly navigate mazes and tracks, I have since had many iterations including gesture-controlled driving with a Raspberry Pi and even a full-sized motorized cart adaptation.
